备注 · 使用说明与名词解释

对外展示或首次阅读时的配套说明：先看「怎么用」，再查「名词」。

怎么用（六个可交互区域）
  1. 首屏结论：先看证据处于历史研究、时间外验证还是前瞻双跑阶段；结论卡下方三条披露与大数字同等重要。
  2. 策略验证：v2 历史高分组与旧引擎买入档按相同覆盖率比较。两边各 68 只，不是“只打 Q5”，更不是“全打基线”。
  3. 固定卡试算：输入截止前可知值，按已注册的 2025 训练边界查 Q1~Q5 与固定卡分数；没有固定买入线，不形成个股收益预测。
  4. 长期核验：先看专家规则从 D1 到 D180 是否衰减，再看完整基石评级、身份构成、集中度和财务质量的独立长期结果；“长期候选”不等于已入生产。
  5. 因子模块与透视：蓝色 IC 表示符合先验，紫色表示与先验相反；红色仅表示负收益。连续因子看五分位，二元/类别因子等待真实分组快照。
  6. 个股数据：桌面端看全指标表，手机端看摘要卡。D1/D5/D30/上市后超额都是上市后的真值，🔒 字段是申购结束后才知道的研究对照，均不能冒充申购时输入。
  7. 建议动线：首屏证据状态 → 策略验证 → 新增基石/稳健性证据 → 固定卡试算 → 孖展快照 → 营收增速 → 个股档案核对。
读懂数字
IC（信息系数，本站指 RankIC）衡量「因子排名与涨幅排名的历史相关程度」，范围 −1 ~ +1。孖展对 D1 的 IC = +0.53，表示本样本中排序关联很强；它不是解释率，也不证明孖展单独造成涨跌。
归一方向（先验方向）每个因子先按常识声明方向（如「发行股数越少越稀缺越好」），统一翻转成「越大越好」后再算 IC。所以 IC 为负 ≠ 算错，而是数据与常识唱反调——估值族因子全部为负就是本轮最大发现。
D1 / D5 / D30上市后第 1 / 5 / 30 个交易日收盘价相对发行价的涨跌幅。D1 = 打新客的首日损益。
上市后超额（对齐口径）D1 收盘→第30个交易日的个股收益，减同一时段恒指收益。它剥离首日跳涨，用来回答首日收盘后还能否继续跑赢。
D180 / D365 首日后超额D1 收盘到第 180 / 365 个交易日的个股收益，减同一时段恒指收益。长期核验优先使用该口径，避免把首日炒作误当成长期质量。
D30 累计超额（旧口径）股票端为发行价→第30个交易日收盘的累计涨幅，再减上市日附近→第30日的恒指涨幅。它包含 D1，且起点并非完全同窗；仅保留作历史累计体验。
破发 / 跑输破发 = D1 < 0；上市后跑输 = 对齐后的 D1→D30 超额 < 0。旧累计 D30 跑输另行标注。
五分位（Q1~Q5）仅适用于连续因子：把股票按归一方向从低到高分五组，Q1 最弱、Q5 最强。二元/类别因子不能伪装成五分位，本页会隐藏不合适的图。
中位数组内一半股票比它好、一半比它差的那个值。新股涨幅极端值多（个别票 +200%），中位数比平均值更抗干扰、更接近「典型体验」。
p 值（置换检验）「这个 IC 纯属运气」的概率。做法：把涨幅标签随机打乱 1000 次重算 IC，真实 IC 若比几乎所有乱序结果都强，p ≈ 0.001。越小越可信，习惯上 < 0.05 算显著。
BH 显著（多重检验校正）同时检验许多因子会产生偶然命中。F2、财务和滚动保荐战绩合并 28 因子统一 BH-FDR 10% 后，15 个仍显著；原始 p 值仍不能单独当作入选结论。
单调（档序 ρ）Q1→Q5 的中位涨幅是否逐级变好：ρ = +1 完美递增，≥ 0.8 判定单调。IC 高但不单调的因子不可信（可能只靠两头极端组撑数字）。
分段 IC（2025H1 / 2025H2 / 2026）把样本按时间切三段分别算 IC，检验因子是否只在某一段行情里灵。入选要求 ≥ 2/3 段为正。
火热期 / 非火热 IC按近两月新股整体表现把市场分冷热两种状态分别算 IC，识别「只有牛市才灵」的伪因子。
市场词汇
孖展（mā zhǎn）港股打新中的融资申购资金。v2 使用申购截止日 12:00 或之前的最后可得快照，并以公开发售货值归一；盘尾最终值和绝对孖展总额只作同源研究参照。
官方最终超购配发结果公告里的公开发售最终认购倍数，截止后才公布，只能作研究对照（🔒）。它与截止前孖展快照在本样本的排名相关 ρ=0.994，但不代表数值校准、尾部误差或信息完全无损。
基石投资者定价前承诺认购并通常锁定 6 个月的投资者。规则版“机构/主权基石股数占比”已在全样本显示显著正向关联并进入现行 cornerstone 维度；零基石、集中度和不透明标签仍只能作风险披露。
关系型基石关联方、客户或供应商身份必须由招股书原文证明。机构名称只能帮助定位，不能独立证明关系；页面显示 0 覆盖代表尚未补齐证据，不代表没有风险。
锁定到期压力锁定期届满时可释放股份相对可交易流通股的比例。必须同时取得锁定天数、释放股数和流通股分母，不能用“基石通常锁六个月”代替个票计算。
保荐人牵头上市申请与尽调的投行。静态“一线/二线/其他”档次已完成别名修正但仍不显著、不单调；近 12 月具体机构战绩方向更合理，却因覆盖不足尚未入卡。
货源归边流通筹码集中在少数持有人手中。它需要配售集中度、可交易流通量等证据；不能仅凭“零基石”直接下结论。
每手金额 / 入场费看板“每手金额”只等于发行价 × 每手股数，不含经纪佣金、平台费及融资成本；它不是券商页面的完整入场费。
A+H 标签当前只有是否 A+H 的二元标记，不能替代玩家真正关心的 AH 折价率。折价率没有合规历史快照前，不作价格高低结论。
招股书财年（组装纪律）财务因子只取申购截止日之前已披露的最后一个完整财年（即招股书里能看到的口径），并做了前视敏感性检验——杜绝「用未来数据预测过去」。
看板标记
⭐ v2 卡在役该因子进入冻结的 v2 双跑卡，不等于单因子全部通过严格筛选：孖展是主因子；发行股数与公开货值是协议注册前声明的近失二级证据。双跑只记账不切生产，前瞻样本到量后终裁。
🔒 截止后可知申购结束后才公布的数据。红线规则（INV-1）：这类字段永远不允许进评分模型，只作研究上限参照。
入选 / 淘汰 / 参照入选 = 三条标准全过（IC 稳定为正 + BH 显著 + 五分位单调）；淘汰 = 至少一条不过（红色 IC 再大也淘汰，因为方向不可解释）；参照 = 不参与评比、只作基准的哨兵因子。
颜色语义蓝 = IC 符合先验或正收益；紫 = IC 与先验相反；红 = 负收益、破发或跑输。散点淡色点表示超出显示范围，悬停看真实值。

方法与口径

  • 样本：2025-01 ~ 2026-07 港股新股 214 只，其中主板 211 只、GEM 3 只（factor_dataset.json，冻结于 2026-07-12）；财务字段 214/214 全覆盖（东财年度指标，全部折算 HKD）。
  • 目标：D1 = 首日收盘/发行价 − 1；上市后超额 = D1 收盘→D30 的个股收益减同一时段恒指收益，是页面默认的后续目标。旧 D30 累计超额从发行价起算、包含 D1 且两端起点不完全同窗，仅保留作历史累计体验。
  • RankIC：Spearman 秩相关，已按先验方向归一（正 = 方向符合先验，负 = 与先验相反）。F2、v3 与保荐滚动战绩合并 28 因子统一做 BH-FDR 10%，15 个保留；孖展 D1 与营收上市后超额又通过按上市月 block 置换。
  • 可知性红线（INV-1）：所有入卡因子必须在申购截止时点可知；「官方最终超购」「定价位置」等截止后数据仅作研究对照。财务因子只取申购截止日前最后一个完整财年（=招股书已披露口径），并做前视敏感性检验（符号全一致）。
  • 连续与类别因子：连续因子看归一 IC、分段稳定、置换检验与五分位单调；二元/类别因子只展示数据层导出的真实标签组，不套用伪 Q1~Q5。估值类即使显著，因方向与先验相反且难以作因果解释，仍被淘汰。
  • v2 因子卡（在役双跑）：孖展快照 80% + 发行股数 15% + 公开发售货值 5%（固定训练窗边界与分值，2025 训练 / 2026 验证）。策略区比较 v2 历史高分组与 expert_v1 旧引擎买入档，在同一验证交集、同一覆盖率下 v2 三项策略指标更好；它不是“全打”对照，也尚未构成前瞻实盘终裁。
  • 基石机构份额：`cs_inst_share` 于 2026-07-14 经用户签字，以 K=16 微调现行 cornerstone 维度 raw 分，维度权重不变；它不属于 DataDriven_v2 的 80/15/5 三因子卡。LLM 六分类未击败规则版，未进入生产。
  • 长期核验：2024–2025 共 181 只；D180 成熟 141 只、D365 成熟 73 只。专家规则历史仿真不是当年生产预测；完整基石评级和基石占发售比例只在 D180 首日后超额通过联合筛选，尚未进入任何长期生产卡。
  • 双跑版本边界：F4 于 2026-07-12 启动，现行引擎在 2026-07-14 加入基石机构份额。因当时前瞻真值仍为 0，运营基线从该提交起固定，并以 rules/code SHA 对账；达到 30 只前不作终裁。
  • 样本偏置：窗口集中在 2025-2026 打新强势期，冷市稳定性仍待前瞻双跑补证；42 只尚缺 D30，其中 40 只集中于 2026 年 6-7 月，因此 D30 分析更偏向较早样本。
  • 风格警示：估值反向与样本期港股成长行情高度耦合，风格切换时可能翻向。页面展示历史关联、分层与验证结果，不证明单一因子导致涨跌，也不构成投资建议。
