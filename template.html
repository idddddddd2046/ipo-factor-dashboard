什么决定一只新股的首日与 30 日？

第一性原理答案：短线是资金现象，中线是资金 + 成长。首日涨跌（D1）几乎完全由申购期抢筹资金决定； 30 日跑赢大盘（D30 超额）由资金热度与营收增速共同驱动；而「便宜」「盈利」在本轮市场不但不提供保护，方向反转。

读懂数字
IC（信息系数，本站指 RankIC）衡量「因子排名能否预测涨幅排名」的分数，范围 −1 ~ +1，0 = 完全没用。经验标尺：|IC| ≥ 0.1 有用、≥ 0.2 很强、≥ 0.5 接近统治级。例：孖展对 D1 的 IC = +0.53 → 申购越火的票，首日大概率涨得越多。用排名相关（Spearman）而非数值相关，因此不被极端值带偏。
归一方向（先验方向）每个因子先按常识声明方向（如「发行股数越少越稀缺越好」），统一翻转成「越大越好」后再算 IC。所以 IC 为负 ≠ 算错，而是数据与常识唱反调——估值族因子全部为负就是本轮最大发现。
D1 / D5 / D30上市后第 1 / 5 / 30 个交易日收盘价相对发行价的涨跌幅。D1 = 打新客的首日损益。
D30 超额D30 涨幅减去同期恒生指数涨幅。剔除大盘水位后衡量「真跑赢」——大盘涨 10% 你涨 8%，其实是跑输。
破发 / 跑输破发 = D1 < 0（首日收盘跌破发行价）；跑输 = D30 超额 < 0（30 日没跑过恒指）。
五分位（Q1~Q5）把全部股票按因子值从低到高排队，均分五组：Q1 = 最低的 20%，Q5 = 最高的 20%。若 Q1→Q5 涨幅逐级抬升、破发率逐级下降，说明因子能有效分层。
中位数组内一半股票比它好、一半比它差的那个值。新股涨幅极端值多（个别票 +200%），中位数比平均值更抗干扰、更接近「典型体验」。
p 值（置换检验）「这个 IC 纯属运气」的概率。做法：把涨幅标签随机打乱 1000 次重算 IC，真实 IC 若比几乎所有乱序结果都强，p ≈ 0.001。越小越可信，习惯上 < 0.05 算显著。
BH 显著（多重检验校正）同时检验 28 个因子，总会有一两个碰巧显著。Benjamini-Hochberg 校正控制这种「测得多、中得巧」的误报，通过者才算真显著。
单调（档序 ρ）Q1→Q5 的中位涨幅是否逐级变好：ρ = +1 完美递增，≥ 0.8 判定单调。IC 高但不单调的因子不可信（可能只靠两头极端组撑数字）。
分段 IC（2025H1 / 2025H2 / 2026）把样本按时间切三段分别算 IC，检验因子是否只在某一段行情里灵。入选要求 ≥ 2/3 段为正。
火热期 / 非火热 IC按近两月新股整体表现把市场分冷热两种状态分别算 IC，识别「只有牛市才灵」的伪因子。
市场词汇
孖展（mā zhǎn）港股打新中向券商借钱申购的杠杆资金（margin 的粤语音译）。孖展认购倍数 = 认购金额 ÷ 公开发售货值，是散户抢筹热度的实时资金投票，申购期内每天可见。
官方最终超购联交所配发结果公告里的公开发售最终认购倍数，申购截止后约 3 天才公布——所以只能当研究对照（🔒），不能进预测模型。好消息：它与截止前可知的孖展快照相关性 0.994，几乎无信息损失。
基石投资者IPO 定价前承诺认购并锁定 6 个月的机构。传统观念是背书加分；2026 年数据显示零基石（货源归边的庄家盘）反而利好首日，方向已修正。
保荐人牵头这单 IPO 的投行。一线 = 高盛/大摩/中金等。实测「一线保荐」对首日涨幅的 IC 为负——顶级投行定价更激进，留给二级市场的空间更小。
货源归边流通筹码集中在少数人手中，抛压小、价格容易被拉动——流通盘占比低、零基石、发行量小都是它的侧写。
招股书财年（组装纪律）财务因子只取申购截止日之前已披露的最后一个完整财年（即招股书里能看到的口径），并做了前视敏感性检验——杜绝「用未来数据预测过去」。

🔒 截止后可知申购结束后才公布的数据。红线规则（INV-1）：这类字段永远不允许进评分模型，只作研究上限参照。
入选 / 淘汰 / 参照入选 = 三条标准全过（IC 稳定为正 + BH 显著 + 五分位单调）；淘汰 = 至少一条不过（红色 IC 再大也淘汰，因为方向不可解释）；参照 = 不参与评比、只作基准的哨兵因子。
蓝色 / 红色（全站统一）蓝 = 正向 / 上涨 / 跑赢；红 = 反向 / 破发 / 跑输。散点图中淡色点 = 涨幅超出 −80%~+200% 显示范围，被截尾在边缘，悬停可看真实值。

方法与口径

  • 样本：2025-01 ~ 2026-07 港股主板新股 214 只（factor_dataset.json，冻结于 2026-07-12）；财务字段 214/214 全覆盖（东财年度指标，全部折算 HKD）。
  • 目标：D1 = 首日收盘/发行价 − 1；D30 超额 = 上市第 1→30 个交易日收益 − 同窗恒指涨幅。破发 = D1 < 0；跑输 = D30 超额 < 0。
  • RankIC：Spearman 秩相关，已按先验方向归一（正 = 方向符合先验，负 = 与先验相反）。显著性 = 标签置换 1000 次（固定种子）+ BH-FDR 10%。
  • 可知性红线（INV-1）：所有入卡因子必须在申购截止时点可知；「官方最终超购」「定价位置」等截止后数据仅作研究对照。财务因子只取申购截止日前最后一个完整财年（=招股书已披露口径），并做前视敏感性检验（符号全一致）。
  • 入选三条标准：全样本归一 IC>0 且 ≥2/3 时段为正；置换检验 BH 显著；五分位单调（档序 ρ≥0.8）。估值类因子即使显著，因方向与先验相反（成长风格暴露、不可解释为因果）仍被淘汰。
  • v2 因子卡（在役双跑）：孖展快照 80% + 发行股数 15% + 公开发售货值 5%（权重∝训练窗 IC，2025 训练 / 2026 验证）；对决基线为专家权重引擎实盘记录，三指标全胜。
  • 风格警示：估值反向与 2025-2026 港股成长行情高度耦合，风格切换时可能翻向——这是它们不入卡的原因。本看板为研究呈现，不构成投资建议。
